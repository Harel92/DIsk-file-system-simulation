Disk file system that simulate unix-fs system with 256 bit of disk memory.
The program works with one main folder and the user can add/delete and open/close files.
For the simulate we will use a txt file to show the disk data as chars.

Run program with "F5" in visual studio code.

Running the the program open's menu with input option of numbers 0-8

0: exit the program and delete the disk
1:show all disk data
2:format the disk (must use this function first to initialize the disk)
3:create file (mush use this before writing to a file)
4:open file 
5:close file (beware! cannot use commands on closed files)
6:write to a file
7:read file data
8:delete file(data still shows in disk until overidden)
